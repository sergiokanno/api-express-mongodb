## Aula-1
    1.  Preparando o ambiente:

        - Instalando Node.js versão 18.16.0
          nvm install 18.16.0

    2.  Criando o servidor

        npm init -y

    3.  Instalando nodemon@3.0.1

        npm install nodemon@3.0.1

    4.  Testando
        
        -   Testando API:

            npm run dev

        -   Acessando rotas:

            http://localhost:3000/
            http://localhost:3000/livros
            http://localhost:3000/autores
